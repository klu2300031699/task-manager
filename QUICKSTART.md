## ğŸ“ Common Tasks

### How to Assign a Task to Someone
1. Click "New Task"
2. Fill in task details
3. In "Assign to" dropdown, select a user
4. Click "Create Task"
5. That user will see it in their "Assigned to me" filter

### How to Change Your Password
1. Click your avatar
2. Go to "Profile" tab
3. Enter current password
4. Enter new password twice
5. Click "Save Changes"

### How to Export Your Tasks
1. Click the floating purple button (bottom-right)
2. Click "Export Tasks"
3. A JSON file will download automatically
4. Save it somewhere safe!

### How to Import Tasks
1. Click the floating purple button
2. Click "Import Tasks"
3. Select your JSON file
4. Tasks will be imported instantly

### How to Clear Completed Tasks
1. Click the floating purple button
2. Click "Clear Completed"
3. Confirm the action
4. All completed tasks are deleted

## ğŸ› Troubleshooting

### Issue: Can't Login
**Solution:** Make sure you're using the correct username (not email) and password

### Issue: Tasks Not Showing
**Solution:** Check your filter settings - you might be on "Completed" filter

### Issue: Can't Assign Task to User
**Solution:** The user must be registered first. Create an account or use demo accounts.

### Issue: Export Not Working
**Solution:** Check browser permissions - downloads might be blocked

### Issue: Theme Not Applying
**Solution:** Refresh the page after changing theme
